JDK环境与项目部署

  • 配置JDK环境变量
    1. 确保JDK的安装路径：/home/ubuntu/application/jdk
    2. 打开配置文件
      1. vim /etc/profile
      2. 修改系统配置，需要管理员（root）权限
      3. 对所用用户生效
    3. 添加配置
      1. export JAVA_HOME=/home/ubuntu/application/jdk/具体jdk解压后的目录名
      2. export PATH=$JAVA_HOME/bin:$PATH
    4. 配置生效
      1. source /etc/profile
    5. 验证
      1. java --version
  • 简单部署SpringBoot项目
    1. 获取项目.jar包，上传至工作目录
    2. 运行：java -jar /home/ubuntu/workpace/my_project.jar
  • docker容器化部署SpringBoot项目
    1. 上传docker-demo.jar至工作目录
    2. 在该目录下创建dockerfile：touch Dockerfile
    3. 对Dockerfile文件配置
      1. vim Dockerfile
      2. 配置如下：
  1. 构建镜像：docker build -t docker-demo:1.0.0 dockerfile所在目录
  2. 查看镜像列表docker images
  3. 创建并运行容器
  1. 查看容器：docker ps -a
