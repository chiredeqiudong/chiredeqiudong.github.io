介绍两种搭建方式：

  • FileBrowser
    • 一个开源的Web文件管理器，它提供了一个可视化的界面来管理您的文件系统
    • 支持预览、上传、下载、编辑和删除文件，方便地创建文件夹、重命名文件和复制、粘贴文件
    • 易于安装和使用，并且可以与其他应用程序进行集成。支持远程管理
  • NextCloud
    • 是一个自由开源的文件同步和共享软件平台，它可以自己的服务器上存储、管理和共享数据
    • 支持在线文档编辑、日历、联系人等
    • 它可以在Windows、Mac OS X和Linux等操作系统上运行，还有Android和iOS的移动应用程序，同样支持远程管理

nextcloud官网

一、软件安装

使用Snap下载和安装Nextcloud极其简化了配置和管理过程，提供了一种便捷、安全且易于维护的部署方式（便捷）

Snapcraft是一个为Linux平台设计的强大软件包管理和打包工具，用于创建、测试和发布Snap软件包

  1. 安装snap、snapcraft
    1. sudo apt update
    2. sudo apt install snap
    3. sudo apt install snapcraft
  2. 安装nextcloud
    1. sudo snap install nextcloud
  3. 查看安装是否成功
    1. sudo snap changes nextcloud
  4. 启动nextcloud
    1. snap start nextcloud

启动成功后，通过域名访问，其默认为80端口

初次安装需要创建管理员账号

二、修改配置

由于nginx默认端口也是80，为了防止冲突，修改nextcloud端口

  • snap set nextcloud ports.http=port

使用snap安装的特点就是方便快捷，基本不需要什么设置。但是snap安装后他的存储目录不太方便找到，不过我们可以对其修改

  • 文件存放目录
    • Nextcloud数据存储所在目录
      • /var/snap/nextcloud/common/nextcloud/data/
    • Nextcloud配置文件所在目录
      • /var/snap/nextcloud/current/nextcloud/config/

修改文件存储位置：

配置文件中的重要参数

  1. trusted_domains：受信任的域名，默认本机ip
  2. datadirectory：存储用户文件的位置，更改时注意新目录权限
  3. db*：数据库信息配置
  4. mail_*：邮件参数设置
  5. overwrite*：代理参数设置
  6. log*：日志相关配置
  7. previews*：信息预览配置
  8. memcache：内存设置
  9. redis：缓存配置

功能很多：

  1. 支持文件共享、上传、下载
  2. 支持在线预览
  3. 挂载其他云存储对象服务
  4. 丰富的插件
