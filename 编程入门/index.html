编程入门

发表于2024-07-20|更新于2024-07-22|团队招新&考核
|字数总计:1.7k|阅读时长:5分钟|阅读量:

适用于大一未接触过编程的大一学生食用

导言

既然填报了计算机类的相关志愿，想必大家或多或少对计算机&编程感兴趣。当然，绝大多数人在高中应该都没有接触过编程，对计算机的了解可能停留于初、高老师讲过的几节微机课。不过，不用担心，0基础恰恰是一次全新的探索，希望这篇文章帮你打开计算机世界的大门。（注意，是打开大门，但是大门之后……）

遇到的问题

对于迫切想要了解编程但毫无头绪的同学，你可能在贴吧、知乎、哔站等平台上搜索过”计算机“、”编程“等相关的信息来尝试入门，不过由于计算机知识非常之多，似乎每个人说的都很要道理，但就是不知道该如何去学！

你可能遇到的问题

  • 学计算机首先你需要掌握一门编程语言，c、c++、java、python、JavaScript等，语言非常之多，无法做出选择

  • 知道了自己要学那一门，那该如何去正确学习呢？

  • 有什么可以帮助我学习的工具吗？

解答

笔者推荐小白入门的话，学习C语言就好；如果你是对编程已经基本了解的话，也可以学习其他语言

  • 理由
    1. 简单易学：C 语言相对于 Java、和 C++ 而言，语法较为简单，没有过多的高级特性和复杂的概念
    2. Python 和 Java 都是基于 C 语言开发的，C++ 是基于 C 语言的扩展，在编程语言基础语法相差不大的前提下，用 C 语言作为编程入门语言是一个好选择
    3. 在 C 和 C++ 中，可以直接使用指针进行编程。理解指针可以帮助你更深入地理解编程语言和计算机底层的工作原理。因为在其他编程语言中或多或少都有指针的概念（不需要自己提前去了解什么是指针，学习C的时候自然会有所接触，到时再了解）
    4. 使用过面向过程的 C 语言，学习 Java、C++ 等面向对象的语言可以更好的理解面向对象的优势（各种语言专长的领域不同，没有孰优孰劣之分）
    5. C/C++ 算法题的题解多；因为 C/C++ 性能高、运行时间短、内存占用小，所以大多数算法比赛的选手更倾向于选择 C/C++；在大多数算法刷题网站中，90%都是C/C++
    6. 如果你是计算机类的相关专业，那么C语言是必修的课程，这也可以为你提前打下基础

绝大多数情况下，视频比书籍更适合小白入门，文档和示例比视频更适合开发者学习

  • C 语言入门视频推荐翁凯的 C 语言课程：浙江大学 翁恺 C 语言入门。最后四集可以不用看，只拿 C 语言当入门语言的话看完结构体就差不多够了
    • 还有其他优秀的 C 语言课程，可自己在哔站寻找
    • 老师好像没有详细的开发工具下载视频讲解，这里我提供了devC++（老师就是用这个的）、VSCode的安装包
      • 网盘链接，提取码：tuzy
      • 具体的安装教程，可以去哔站搜，也可以直接在文档上找
  • 书籍（尤其是入门的书籍）可以当作工具辅助书。预习、复习和查阅的时候使用是最好的，别一开始就拿书来啃
  • 一个方面最多买一本书就行了，不要傻不拉几的买好几本，然后放在角落吃灰
  • 远离清华大学出版社等 xx 大学出版社出版的书，绝大部分质量极差
  • 推荐异步图书、图灵系列的图书，推荐人民邮电出版社和机械工业出版的图书，推荐外国的书

对于辅助我们学习的工具有很多，这里主要推荐如何做笔记，这对入门来说很重要！

语言是不是学的越多越好？

  • 我们前面已经讨论过了为什么要先学C

  • 我们学习语言，是因为要用语言对应的技术去开发产品。就目前而言：

  • AI 方面的基本都使用 Python

  • 嵌入式开发使用 C 语言

    嵌入式系统开发是对于除了电脑之外的所有电子设备上操作系统的开发，开发对象有手机，掌上电脑，机电系统等 (from 百度百科)

  • 前端先学 HTML、CSS、JavaScript 三大件 (现在还要学 TypeScript)，然后 vue 或 react

  • 后端绝大多数为 Java，少量大公司如字节跳动、哔哩哔哩、知乎等用 Go，也有公司是用 Python，不过相对更少

  • 游戏服务器几乎都是 C++

  • 大数据领域，Java 和 Python 平分秋色，开发者主要是使用它们来调用另一种语言——SQL

  • 各方向的详细介绍和分流我们会在开学后进行，到时还会有专门的宣讲会进行线下讲解，在这里只是做简单的介绍
